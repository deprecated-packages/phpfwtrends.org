How is it Measured?

Does Laravel downloads boost Symfony?

Many Laravel packages are using Symfony packages as base structure. It would not be fair for Laravel to add these downloads to Symfony, as Laravel users don't use Symfony directly. To avoid this false positive, Laravel downloads are removed from Symfony total downloads.

1 download of Laravel package = 0 download of Symfony packages. See list of packages cleared this way.

What Packages are Excluded?

  • packages with < 500 downloads daily - not relevant in this scale
  • packages younger < 12 months, so we're able to compute trend
  • sandbox packages or meta packages that have nothing to do with framework usage, e.g. symfony/demo - see list of these packages
